# Number Converter Android App

A simple Android application that converts numbers between decimal, binary, octal, and hexadecimal systems.

## Features
- Clean Material Design UI
- Real-time number conversion
- Input validation
- Error handling

## Technologies Used
- Java
- Android SDK
- XML Layouts
- Gradle Build System

## Installation
1. Download the APK from releases
2. Install on Android device
3. Or import project in Android Studio
